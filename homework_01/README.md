Цель:
Написать телефонный справочник, который будет сохранять контакты в файл и иметь следующий функционал:

открыть файл
сохранить файл
показать все контакты
создать контакт
найти контакт
изменить контакт
удалить контакт
выход

Исходный файл справочника:
phone_book.txt
используется разделитель ";"
разместь в оном проекте (папке) с исполняемым файлом main.py

в начале работы со справочником выполните пунк 1 "открыть файл"
